== Description ==

The Brad's Google Analytics plugin allows a user to add the Google Analytics Tracking Code to a WordPress website. That is all it does, and that is all it will ever do.

If you like this plugin, please consider making a small donation at http://mehder.com/donate/.
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Brad's Google Analytics to add the Google Analytics Tracking ID
